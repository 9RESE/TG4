# How-To Guides

Problem-solving recipes for specific tasks in TG4.

## Overview

How-to guides are task-oriented. They take you through the steps to solve real-world problems. Unlike tutorials, they assume you already have basic knowledge.

## Guides

### Configuration

| Guide | Description |
|-------|-------------|
| Configure API Keys | Set up Kraken and LLM provider credentials |
| Adjust Risk Parameters | Modify stop-loss, position sizing, leverage |
| Set Up Alerts | Configure Grafana alerts for trading events |

### Operations

| Guide | Description |
|-------|-------------|
| Start/Stop Trading | Safely start and stop the trading system |
| Switch Trading Pairs | Change which pairs are actively traded |
| Export Trade History | Export trades for tax or analysis |

### Troubleshooting

| Guide | Description |
|-------|-------------|
| Debug WebSocket Issues | Diagnose connection problems |
| Recover from Errors | Handle and recover from system errors |
| Performance Tuning | Optimize system performance |

## Guide Structure

Each how-to guide follows this format:

1. **Goal**: What you'll accomplish
2. **Prerequisites**: What you need to know/have
3. **Steps**: Numbered instructions
4. **Verification**: How to confirm success
5. **Common Issues**: Troubleshooting tips

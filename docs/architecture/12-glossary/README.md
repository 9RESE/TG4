# 12 - Glossary

## Trading Terms

| Term | Definition |
|------|------------|
| OHLCV | Open, High, Low, Close, Volume - standard candle data |
| Stop-Loss | Order to close position at specified loss level |
| Take-Profit | Order to close position at specified profit level |
| Drawdown | Peak-to-trough decline in portfolio value |
| Sharpe Ratio | Risk-adjusted return metric |
| Leverage | Borrowed funds multiplier |
| Margin | Collateral for leveraged positions |
| Slippage | Difference between expected and executed price |
| Spread | Difference between bid and ask price |
| Liquidity | Ease of buying/selling without price impact |

## Technical Terms

| Term | Definition |
|------|------------|
| WebSocket | Full-duplex communication protocol |
| TimescaleDB | PostgreSQL extension for time-series data |
| LLM | Large Language Model |
| EMA | Exponential Moving Average |
| RSI | Relative Strength Index |
| Bollinger Bands | Volatility indicator using standard deviations |
| MACD | Moving Average Convergence Divergence |

## System Components

| Term | Definition |
|------|------------|
| TG4 | TripleGain - project codename |

| Risk Manager | Component enforcing risk limits |
